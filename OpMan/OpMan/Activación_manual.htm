Activación manual

El sistema contra incendios está formado por una serie de tubos en la instalación que distribuyen CO2 al activarse los sensores de seguridad contra incendios. Estos sensores se podrán encontrar justo debajo de las boquillas.

Si falla el sistema automático, el sistema se podrá accionar mediante un interruptor con tapa de vidrio rompible. También hay un tirador manual para la liberación del CO2.



Mensajes de Advertencia Importantes

 

Si se activa manualmente el sistema y éste falla por cualquier motivo, se deberán llenar nuevamente los cilindros de CO2 .
